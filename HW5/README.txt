Создать класс User в модуле Github (все задание должно быть выполнено в одном файле).

Пользуясь Github API (http://developer.github.com/v3/), решить следующие задачи:

Задание 1.

Написать метод класса all_users, который возвращает массив логинов пользователей.

Задание 2.

2.1. Создать аксессор name для пользователя. 
2.2. Создать метод info, который будет возвращать информацию о пользователе с заданным ником (имя в поле name). Информация должна быть представлена в виде руби-хеша.
2.3. Создать метод id, который возвращает id пользователя с заданным ником.

Задание 3.

3.1. Создать метод repos, который возвращает информацию о репозиториях пользователя с заданным ником.
3.2. Создать метод repos_names, который возвращает массив названий репозиториев данного пользователя.
3.3. Создать метод total_repos_size, который считает общий размер репов данного пользователя
3.4. Создать метод prefferable_languages, который показывает любимые языки пользователя (т.е. то, на чем он пишет). Эти языки хранятся в информации о репозиториях.

В файле main.rb собраны вызовы всех функций, которые необходимы для дз.

Количество запросов на гитхаб ограничено до 60 в час.
